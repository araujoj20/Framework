OSEE_STATIC_INLINE OsEE_CDB * osEE_get_core ( CoreIdType core_id)

The function returns the pointer to the CDB of the core specified as parameter. Since the machine is a single core machine, the function returns the only CDB available.

Parameters
[in]core_idThe core for which we should return the information
Returns
Always returns the only CDB available (it's a single core machine)
