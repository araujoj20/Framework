struct OsEE_CTX_tag* OsEE_CTX_tag::p_ctx

In the case of ERIKA3, every time a context is saved, and a new task is stacked on top of it, we need to remember the previous context, because in case of a TerminateTask() we need to unwind the stack and recover the previous context
