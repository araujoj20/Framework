\doxysection{Implemented Standards }
\hypertarget{implemented-standards}{}\label{implemented-standards}\index{Implemented Standards@{Implemented Standards}}
The ERIKA3 Operating System provides an operating system interface which implements the following standards\+:


\begin{DoxyItemize}
\item OSEK/\+VDX specification, version 2.\+2.\+3
\item which has been later on standardized as ISO 17356-\/3
\item with the extensions of AUTOSAR OS 4.\+3
\end{DoxyItemize}

Currently (version GH55), ERIKA3 implements all features specified by the AUTOSAR OS Scalability Class 1.
