\doxysubsection{Detailed Description}
The descriptor block tracks the specific mutual exclusion object, which can be either a Resource or a Spin Lock. They are handled together because they have to be released in LIFO order \mbox{[}SWS\+\_\+\+Os\+\_\+00801\mbox{]}.
